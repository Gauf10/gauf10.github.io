# Gabriel Aufgang — Mentor Landing Page

Landing page personal para presentar mi trabajo como mentor, speaker y director creativo.

El sitio busca transmitir una mirada humana y estratégica sobre creatividad, liderazgo, emprendimiento y desarrollo de proyectos, priorizando claridad, identidad y conexión genuina.

## Sobre el proyecto

Esta web funciona como un espacio de presentación y conexión para:

- mentorías
- charlas y workshops
- acompañamiento a emprendedores y equipos
- proyectos creativos
- exploraciones vinculadas a comunicación, liderazgo e improvisación

La idea no es solo mostrar experiencia, sino también reflejar una forma de pensar y acompañar procesos.

## Enfoque

El proyecto está construido alrededor de algunos conceptos clave:

- buenas preguntas
- creatividad aplicada
- pensamiento estratégico
- improvisación
- crecimiento sostenible
- perspectiva a largo plazo
- conexión humana

## Objetivo

Crear una experiencia simple, clara y cercana, donde el contenido tenga protagonismo y la navegación se sienta liviana e intuitiva.

## Estado

Proyecto en evolución constante.
